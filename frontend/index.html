      User Guide
      
            
      Smart Expense Workflow Guide
      
            
      Learn how to track expenses, manage budgets, monitor savings, and understand your financial health.
      
          
      
          
      
            
            
            
          
      
        
      

        
      
          
      Set Budget

      01
      
              
      Getting Started
      
              
        
                
      • Open the Dashboard to view your monthly financial summary.
        • 
                
      • Select the dashboard month.
        • 
                
      • Review financial health score, budget used, actual spending, savings, and quick insights.
        • 
              
      
            
      
          
      

          
      
            
            
      
              
      02
      
              
      Add Expenses
      
              
        
                
      • Go to the Expenses page or click Add Expense.
        • 
                
      • Enter amount, category, date, payment method, and notes.
        • 
                
      • Save the transaction so dashboard cards and charts update automatically.
        • 
              
      
            
      
          
      

          
      
            
            
      
              
      03
      
              
      Manage Budget
      
              
        
                
      • Go to the Budget page.
        • 
                
      • Add or update the monthly budget.
        • 
                
      • Allocate budget across Home, Food & Groceries, Utilities & Bills, Transport, and Miscellaneous.
        • 
                
      • Track used budget and remaining budget.
        • 
              
      
            
      
          
      

          
      
            
            
      
              
      04
      
              
      Review Insights
      
              
        
                
      • Go to the Insights page.
        • 
                
      • Review AI-generated spending insights.
        • 
                
      • Identify high-spending categories and spending trends.
        • 
                
      • Use insights to control unnecessary expenses.
        • 
              
      
            
      
          
      

          
      
            
            
      
              
      05
      
              
      Check Forecast
      
              
        
                
      • Go to the Forecast page.
        • 
                
      • Review projected month-end spending.
        • 
                
      • Check the displayed forecast confidence.
        • 
                
      • If fewer than 7 days of data are available, treat the forecast as an early estimate.
        • 
              
      
            
      
          
      

          
      
            
            
      
              
      06
      
              
      View Reports
      
              
        
                
      • Go to the Reports page.
        • 
                
      • Generate monthly finance reports.
        • 
                
      • Review total spending, remaining budget, savings, top categories, insights, and forecast summary.
        • 
              
      
            
      
          
      

          
      
            
            
      
              
      07
      
              
      Manage Profile
      
              
        
                
      • Go to the Profile page.
        • 
                
      • Manage user details and preferences.
        • 
              
      
            
      
          
      

          
      
            
            
      
              
      Monthly Routine
      
              
      Recommended Monthly Workflow
      
              
      Set Budget -> Add Expenses -> Track Spending -> Review Insights -> Check Forecast -> Generate Report -> Improve Savings.
